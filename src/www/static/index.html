Demo

  • Get a variant from a ID:

    http://dev2.mygene.info:8000/api/variant/<variant_id>

    • http://dev2.mygene.info:8000/api/variant/1:g.35367C>T
    • http://dev2.mygene.info:8000/api/variant/7:g.55241707G>T
    • http://dev2.mygene.info:8000/api/variant/9:g.107620835C>T

  • Making queries (query syntax):

    http://dev2.mygene.info:8000/api/query/?q=<query>

    • query for anything

      http://dev2.mygene.info:8000/api/query?q=rs71409357

    • query for a genomic range

      http://dev2.mygene.info:8000/api/query?q=chr1:69000-70000

    • query for one or more fields

      http://dev2.mygene.info:8000/api/query?q=dbsnp.snpclass:single

    • support wild char

    • support regex

    • boolean query (AND, OR, NOT)

      http://dev2.mygene.info:8000/api/query?q=(NOT func:unknown)

    • faceting

      http://dev2.mygene.info:8000/api/query?q=(NOT func:unknown)&facets=func&size=0

    • field exists

      http://dev2.mygene.info:8000/api/query?q=_exists_:dbsnp AND _exists_:cosmic
