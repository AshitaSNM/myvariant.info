Introduction

This is a project coming out of 1st NoB Hackathon.

The scope of this project is to aggregate existing annotations for genetic variants. Variant annotations have drawn huge amount of efforts from researchers, which made many variant annotation resources available, but also very scattered. Doing integration of all of them is hard, so we want to create a simple way to pool them together first, with high-performance programmatic access. That way, the further integration (e.g. deduplication, deriving higher-level annotations, etc) can be much easier.

Code repo: https://github.com/Network-of-BioThings/myvariant.info

Demo

  • Get a variant from a ID:

    http://myvariant.info/api/variant/<variant_id>

    • http://myvariant.info/api/variant/chr1:g.35367C>T
    • http://myvariant.info/api/variant/chr7:g.55241707G>T
    • http://myvariant.info/api/variant/chr9:g.107620835C>T
    • http://myvariant.info/api/variant/chr1:g.160145947G>T
    • http://myvariant.info/api/variant/chr16:g.28883241A>G
    • http://myvariant.info/api/variant/chr3:g.49721532C>T

  • Making queries (query syntax):

    http://myvariant.info/api/query/?q=<query>

    • query for any supported id (currently rsid, will include more as we go)

      http://myvariant.info/api/query?q=rs58991260

    • query for a genomic range

      http://myvariant.info/api/query?q=chr1:69000-70000

    • query for one or more fields

      http://myvariant.info/api/query?q=dbsnp.snpclass:single

    • support wild char

    • support regex

    • boolean query (AND, OR, NOT)

      http://myvariant.info/api/query?q=(NOT func:unknown)

      • http://myvariant.info/api/query?q=dbnsfp.polyphen2.hdiv.score:>0.99 AND chrom:1

    • faceting

      http://myvariant.info/api/query?q=(NOT func:unknown)&facets=func&size=0

    • field exists

      http://myvariant.info/api/query?q=_exists_:dbsnp AND _exists_:cosmic
